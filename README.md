# Sistema-IoT---monitoreo-de-variables-ambientales
Panel web y firmware para un sistema de monitoreo de calidad del aire basado en Wemos D1 R32, con el uso de los sensores SCD40 (CO2, temperatura, humedad) y MPM10-AG (PM1.0/PM2.5/PM10).
